# Лабораторная работа 2: Работа со структурами в C

## Overview
Данный проект реализует программу на языке C для работы со структурой STUDENT. 
Программа позволяет:
- Вводить данные о студентах (фамилия, курс, успеваемость)
- Сортировать записи по алфавиту
- Находить студентов с средним баллом выше общего среднего
- Выводить результаты на экран

## Usage
### Компиляция и запуск:
```bash
# Сборка проекта
make

# Запуск программы
make run

# Очистка временных файлов
make clean
```
## Additional Notes
Проект использует модель КИС: main.c (клиент), заголовочные файлы в include/ (интерфейсы), реализации в src/ (серверы). Настроен GitHub Actions для автоматической сборки.
